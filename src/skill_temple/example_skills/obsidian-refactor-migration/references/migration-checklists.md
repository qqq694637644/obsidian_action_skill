# Migration Checklists

## 1. 重命名前检查

- 是否已有同名文件或同名别名？
- 是否存在同名异物？
- 旧名称是否仍有检索价值，需要转 alias 吗？
- 关键索引页、MOC、Base、Dataview、Canvas 是否引用此名称？

## 2. 拆分前检查

- 是否真的存在多个独立实体或生命周期？
- 原笔记是否还能承担索引角色？
- 拆分后的每个子笔记是否有清晰名称与回链？

## 3. 归档前检查

- 是否还有未完成任务？
- 是否仍被 active dashboard / base / query 使用？
- 是否需要保留结项总结、时间范围、归档原因？

## 4. 属性迁移前检查

- 新字段名是否已在别处被占用为别的语义？
- 字段类型是否一致？
- 模板、搜索、Bases、Dataview、脚本是否依赖旧字段？
- 是否需要共存期和回滚说明？
